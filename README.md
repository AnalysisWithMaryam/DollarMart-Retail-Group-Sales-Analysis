# 📊 DollarMart Sales-Analysis

### Project Overview
This Project analyzes sales data from DollarMart Retail Group to help stakeholders understand key metrics such as total revenue, product performance, customer behaviour and regional sales trend.

### Tools and Techniques
* Microsoft excel
* Pivot tables
* Pivot Charts
* Slicers for interactive filtering
* Data cleaning and transformation
* Data visualization
* KPI calculations

 ### Dataset Overview
  Order ID, Order Date, Product Name, Product Category, Region, Sales Revenue, Quantity Sold

 ### Data Cleaning Process
 * Checked for missing or incomplete values
 * Removed duplicate records
 * Standardized date formats
 * Created calculated columns for revenue related metrics
 * Ensured consistent category and region naming

 ### Dashboard KPIs
 * Total Unit Sold
 * Total Sales Revenue
 * Average Revenue per Customer
 * Average Revenue per Product

 ### Excel Dashboard
 The excel dashboard includes the following visuals:
 * Year-over-Year Revenue Comparison (2023 vs 2024)
 * Monthly Revenue Trends
 * Revenue Comparison (Weekends vs Weekdays)
 * Top 5 Revenue Generating Products
 * Revenue by Age Segment
 * Revenue Distribution by Gender
 * Regional Sales Performance
